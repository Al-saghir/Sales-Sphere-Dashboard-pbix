# Sales Sphere Dashboard (Power BI)

## 📊 Overview
Sales Sphere Dashboard is an interactive Power BI project designed to analyze sales performance and provide actionable business insights. The dashboard transforms raw data into meaningful visualizations that support data-driven decision-making.

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Excel (Data Source)

## 🚀 Features
- Dynamic KPI tracking (Revenue, Profit, Growth)
- Advanced DAX measures for accurate calculations
- Interactive filters and slicers
- Drill-down analysis across time and categories
- Clean and user-friendly dashboard design

## 📈 Key Insights
- Identification of top-performing products and regions
- Analysis of customer segments and purchasing behavior
- Detection of sales trends and seasonal patterns

## 📂 Project Structure
- `Sales Dashboard.pbix` → Power BI file
- `Screenshots` → Dashboard previews

## 🎯 Outcome
- Improved reporting efficiency
- Reduced manual analysis effort
- Enabled faster and more accurate business decisions
